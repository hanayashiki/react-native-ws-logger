# react-native-ws-logger

An out-of-the-box websocket network viewer

## Installation

```sh
npm install react-native-ws-logger
```

## Usage

```js
import WsLogger from "react-native-ws-logger";

// ...

const result = await WsLogger.multiply(3, 7);
```
